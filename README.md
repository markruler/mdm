# Markdown Mail

Command `>mdm: Send markdown to mail` sends markdown file to mail.

## Front Matter

```markdown
---
host: mail.example.com
port: 25
from: from@example.com
to: to@example.com
subject: This is email subject
---

# Markdown Mail
```

## Contributing

```sh
code mdm
```

```sh
npm install
```

In VS Code, press `F5` to open a new window with your extension loaded.
